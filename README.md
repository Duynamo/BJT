# 🇯🇵 BJT Flashcard Web App - Chinh phục Tiếng Nhật Thương Mại 🚀 

![GitHub License](https://img.shields.io/github/license/DuyDynamo/BJT)
![GitHub stars](https://img.shields.io/github/stars/DuyDynamo/BJT?style=social)
![Build Status](https://img.shields.io/badge/Status-Live-success)

**BJT Flashcard Web App** là công cụ hỗ trợ học tập thông minh, được thiết kế dành riêng cho những ai đang ôn luyện kỳ thi **Business Japanese Proficiency Test (BJT)**. Ứng dụng tập trung vào việc tối ưu hóa trí nhớ thông qua Flashcards và tự động hóa dữ liệu từ vựng.

🌐 **Trải nghiệm ứng dụng tại:** [https://duynamo.github.io/BJT/](https://duynamo.github.io/BJT/)

---

## ✨ Điểm nổi bật (Key Features)

- 🗂️ **Hệ thống Flashcards Thông Minh:** Giao diện trực quan, hỗ trợ lật thẻ ghi nhớ từ vựng, cách đọc (Furigana) và ý nghĩa trong ngữ cảnh kinh doanh.
- ⚙️ **Dữ liệu Tự Động Hóa:** Quy trình trích xuất từ vựng tự động từ các tài liệu PDF/CHM sang định dạng Markdown/JSON để nạp vào hệ thống.
- 📱 **Mobile First:** Giao diện phản hồi (Responsive), hỗ trợ học tập mọi lúc mọi nơi trên điện thoại và máy tính.
- ⚡ **Hiệu Năng Cao:** Web app thuần HTML/CSS/JS, tốc độ tải cực nhanh và hoạt động ổn định.

---

## 🛠️ Công nghệ sử dụng (Tech Stack)

- **Frontend:** HTML5, CSS3 (Modern UI), JavaScript (ES6+).
- **Automation/Data:** Python (Sử dụng để xử lý dữ liệu từ vựng từ các nguồn tài liệu BJT).
- **Deployment:** GitHub Pages.
- **Workflow:** Git & GitHub (CI/CD cơ bản).

---

## 📖 Quy trình phát triển (Workflow)

Dự án này không chỉ là một ứng dụng Web mà còn là kết quả của sự kết hợp giữa kỹ thuật lập trình và tự động hóa:
1. **Extraction:** Sử dụng Python script để parse dữ liệu từ các tệp tài liệu tiếng Nhật.
2. **Transform:** Chuyển đổi dữ liệu thô sang cấu trúc JSON thân thiện với ứng dụng.
3. **Display:** Render dữ liệu lên giao diện người dùng tối giản, tập trung vào sự tập trung (Deep Learning).
